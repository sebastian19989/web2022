Navegadores web

Un navegador web (en ingles,web browser)es un software,aplicacion o programa que permite el acceso a la web, interpretando la informacion de distintos tipos de archivos y sitios web para que estos puedan ser vistos.

La funcionalidad basica de un navegador web es permitir la visualizacion de documentos de texto, posiblemente con recurosos multimedia incrustados

Ejemplos de navegadores web

safari:safari es un navegador web de codigo cerrado desarrollado por Apple inc.Esta disponible para macOS,ipad e iOS.

Google Chrome: es un navegador web de codigo cerrado desarrollado por Google, aunque derivado de proyectos de codigo abierto.

Mozilla firefox es un navegador web libre y de codigo abierto desarrollado para linux,Android,iOS,macOS y Microsoft Windows coordinado por la corporacion Mozilla y la fundacion Mozilla.
